# Publicar Caga2Dos Atlas en GitHub Pages

1. Crea o abre el repositorio.
2. Sube **el contenido de esta carpeta**, no la carpeta entera.
3. En la raíz deben verse `index.html`, `assets`, `.nojekyll` y `404.html`.
4. Ve a **Settings → Pages**.
5. En **Build and deployment** elige:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/(root)**
6. Pulsa **Save** y espera 1–3 minutos.
7. Abre la dirección que GitHub muestra en esa misma pantalla.
